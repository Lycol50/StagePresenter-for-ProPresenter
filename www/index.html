STAGEPRESENTER
for ProPresenter

A better Stage Display for ProPresenter 7

A Stage display that is automatically synchronized with ProPresenter 7 and displays Presentations, Slides, Playlists, Timers, Countdowns and Stage-Messages. StagePresenter shows far more than only the current and next slide.

StagePresenter can be used on the same machine, where ProPresenter is running. But it can also be installed on a different machine on the same network.

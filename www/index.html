STAGEPRESENTER
for ProPresenter

A better Stage Display for ProPresenter 7

The Stage Display is automatically synchronized with ProPresenter 7 and displays Presentations, Playlists, Slide-Previews, Timers, Countdowns and Stage-Messages.

StagePresenter can be used on the same mac, where ProPresenter is running. But it can also be installed on a different computer on the same network.
